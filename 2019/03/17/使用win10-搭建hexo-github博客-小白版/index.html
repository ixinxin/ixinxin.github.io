使用win10-搭建hexo-github博客-小白版

这是我的第一篇博客希望能开个好头，给大家分享一下如何搭建heso-GitHub博客，在此期间身边的朋友都很快搭建好了而我一直卡着很不顺利，但所幸在学长的帮助下最终也是完成了。

前言

想要搭建hexo-GitHub博客首先要有一个GitHub账号这篇博客是以纯小白，傻瓜式操作视角来操作的，但也因作者水平有限，其中有上面不足还请大家指点交流。

第一步：注册一个GitHub账号

登陆GitHub官方网站
点击注册: Writing，
按照要求一步步完成，如果看不懂使用谷歌浏览器自带翻译。
值得注意的是第二步会让你选择这时会弹出一个界面，让你选择你的私人计划（personal plan），即选择免费用户还是付费用户。付费用户可以拥有私人代码仓库（repos），即别人不能查看你的代码。免费用户的仓库都是公开的，任何人都能查看。
这里我们选择免费用户就可以了。

后面是一个问卷调查直接跳过即可。
完成注册后会让你验证邮箱，我们直接进入直接所填写的邮箱找到GitHub给我们发生邮件，打开点击里面的连接就会跳转到注册完成的界面.

第二步：搭建环境

安装node.js、npm，安装git for windows（需要翻墙下载）

（1）在浏览器上搜索node.js，进入官网：Writing点击左边的方块
下载完成后打开cmd（此时NPM已经捆绑安装）

若如此显示则安装完成

(2)在浏览器上搜索git for windows，进入官网:Writing点击左边的方块。

打开一直next直到出现下图选择第二个

下载完成后打开cmd

若如此显示则安装完成。
