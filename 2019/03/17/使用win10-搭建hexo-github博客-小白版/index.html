使用win10-搭建hexo-github博客-小白版

这是我的第一篇博客希望能开个好头，给大家分享一下如何搭建heso-GitHub博客，在此期间身边的朋友都很快搭建好了而我一直卡着很不顺利，但所幸在学长的帮助下最终也是完成了。

前言

想要搭建hexo-GitHub博客首先要有一个GitHub账号这篇博客是以纯小白，傻瓜式操作视角来操作的，但也因作者水平有限，其中有上面不足还请大家多多指点交流。

第一步：注册一个GitHub账号

登陆GitHub官方网站
点击注册:，
按照要求一步步完成，如果看不懂使用谷歌浏览器自带翻译。
值得注意的是第二步会让你选择这时会弹出一个界面，让你选择你的私人计划（personal plan），即选择免费用户还是付费用户。付费用户可以拥有私人代码仓库（repos），即别人不能查看你的代码。免费用户的仓库都是公开的，任何人都能查看。
这里我们选择免费用户就可以了。

后面是一个问卷调查直接跳过即可。
完成注册后会让你验证邮箱，

我们直接进入直接所填写的邮箱找到GitHub给我们发生邮件，打开点击里面的连接就会跳转到注册完成的界面，此时完成验证。

安装node.js、npm，安装git for windows（需要翻墙下载）

    下载完成后打开cmd（此时NPM已经捆绑安装）

第三步：建库与配置密钥

  1. 建立仓库

    进入github官方登陆账号
    点进去会显示如下界面，新建一个名为你的XXX.github.io的仓库，比如说，如果你的github用户名是XXX，那么你就新建XXX.github.io的仓库（必须是你的用户名，其它名称无效），将来你的网站访问地址就是 http://XXX.github.io 了。
    此处如果没有验证邮箱，否则不会成功

第四步：使用hexo写博客

进入cmd

在电脑的某个地方新建一个名为hexo的文件夹（命名为你的用户名），比如我的是D:\ixinxin\hexo，由于这个文件夹将来就作为你存放代码的地方，所以最好不要放C盘。

等待一段时间会得到下图

INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

当你看到这样的回答时，你的博客在本地已经成功搭建，你可以在浏览器中输入http://localhost:4000 ，或127.0.0.1：4000来访问自己的博客。

更换主题

刚搭建好的博客是默认主题，hexok也为我们自动写了一篇Hollywood，当然大家对此肯定也不会多满意，此时首先来替换一个好看点的主题。这是官网主题选择自己喜欢的主题下载到

修改_config.yml（在hexo目录下）中的theme: landscape改为theme: yilia，然后重新执行hexo g来重新生成。
如果出现一些莫名其妙的问题，可以先执行hexo clean来清理一下public的内容，然后再来重新生成和发布。
